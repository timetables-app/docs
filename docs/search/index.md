# Komponent `search`

## Odpowiedzialność
Komponent dostarcza algorytm wyszukiwania bazujący na strukturach
i danych dostarczonych przez komponent `schedule`.

## Zależności
Komponent najwyższego poziomu - zależy od `schedule` i `community`
